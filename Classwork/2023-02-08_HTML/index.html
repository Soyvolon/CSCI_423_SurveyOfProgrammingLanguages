JavaScript is the foundation of browsers and does not seem to be going away anytime soon. It's much integrated into our browsers to simply disappear, and even as new applications move forward with web development, they still rely heavily on JavaScript for interaction with the displayed website. In the case of SPAs, web delivered or not, they send their own JS bundle over to the client to facilitate interaction.

As much as I detest using Python for anything beyond a non-object oriented script to do some string processing or similar, it is a popular language. On top of that, it's easy to use, and thats attracting a lot of people to the language. New students, existing professors, and professionals are all using Python and I don't think that is going to change. It may not be the most popular, but I don't think it's going to leave the top of the charts anytime soon. Especially not with the packages and tools used for data science in Python - plenty of people are using those as well.

As my personal favorite language, I may be biased in this, but I do believe C# will stick to the tops of the charts. The .NET environnement has gone completely cross platform at this point, and it's open source. There is a lot of variety in what you can do with the language - from web applications to desktop to mobile apps - and the documentation is phenomenal. Maybe my favorite language wont top the charts in ten years, as much as I hope it will, but it's not going away.
